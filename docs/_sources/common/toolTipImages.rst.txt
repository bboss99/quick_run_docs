
Tooltip Images
================

Steps to create tooltip images without writing any code.


It's easier to understand using following tree

.. code-block::

    scripts/
        create_cubes.py
        _tti/
            create_cubes.py.png


Lets say, you have a file named ``create_cubes.py`` inside a directory ``scripts``.

You can create a folder ``_tti`` in that directory,  and save a file named named ``create_cubes.py.png``
inside ``_tti`` directory.

now you have an image tooltip for ``create_cubes.py`` file.

.. note::

    parameters you see while creating ui.

    tti: tool tip image

    ttt: tool tip text
