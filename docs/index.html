Welcome to Quick Run’s documentation!¶

This is an addon which will help you run scripts. This is very simple and doesn’t require much explanation. You save a python file in a folder, then run the operator and you see a button for that file. Script may contain Boss UI code which makes it a lot powerful.

Note

You should read this documentation along with Boss

Note

Code in this documentation have been written keeping in mind a artist who can write some python, meaning a beginner python coder. If you are intermediate level python user/programmer, you may find explanations boring and unnecessary.
