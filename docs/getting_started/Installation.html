Download/Installation¶

Download¶

You can get quick run addon from here Quick Run Free and Quick Run.

Installation¶

Installation is two step process. Hopefully you have done the first first step of putting boss package in the modules folder as described here .

If not, above link also contain boss directory, just place it in correct folder as described here.

Note

You can install add-on like any other blender add-on (Preference -> addons -> install).
